# 検証手順: 逆転の疾走（GRAVITY SHIFT） プロトタイプ (Phase 3)

以下の手順でフェーズ3（ゲームサイクル）の動作確認をお願いします。

## 1. プロジェクトの起動
1. Godot 4.5.1 で `e:\godot\gragra` プロジェクトを開いてください。
2. `scenes/Main.tscn` を開き、**F6キー** で実行します。

## 2. 動作確認項目

### 新機能: 挙動調整 [IMPACT FOCUS]
- [ ] **死亡時カメラフォーカス**:
    - スパイクにぶつかった瞬間、カメラがキャラクターの中心ではなく、**「ぶつかった場所（スパイクの場所）」にズーム**することを確認してください。
    - 時間が動き出すと、カメラが引きながら元の位置（キャラクターセンター）に戻ることを確認してください。

### 既存項目の再確認
- [ ] **完全破壊時のカメラ**: 地面に留まる。
