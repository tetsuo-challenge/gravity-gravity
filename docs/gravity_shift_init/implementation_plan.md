# 実装計画: テスト用ギミック追加

検証のために、以下の要素を追加・配置します。

## 1. 重力落下スパイク (FallingSpike)
- **概要**: プレイヤーと同じく重力の影響を受け、重力反転時に天井（または床）へ落ちていくスパイク。
- **構成**:
  - `RigidBody2D`: 物理挙動を担当。`gravity_sensitive` グループに所属。
  - `Area2D` (子ノード): 接触ダメージ判定用。
  - `CollisionShape2D`: 物理衝突用。
- **挙動**: 重力反転時、`Main` または `Player` からの `call_group` を受けて `gravity_scale` が反転する。

## 2. 空中台座 (Floating Platform)
- **概要**: 空中に浮いている足場。重力反転して天井走行から飛び移るなどのテストに使用。
- **実装**: `Main.tscn` 内の `Level` ノードに `StaticBody2D` として配置。

## 3. 天井チェックポイント (Ceiling Checkpoint)
- **概要**: 天井に配置されたチェックポイント。重力反転しないと触れられない。
- **実装**: `Checkpoint` インスタンスを Z軸(180度)回転させて配置。

## テストコース配置案 (Main.tscn)
現在のテストエリア (X=1500以降) を拡張:
- **X=1800**: 空中台座（床からも天井からも届かない中間位置）。
- **X=2000**: 天井チェックポイント。
- **X=2200**: 落下スパイク群（重力を変えると襲ってくる、または道を開ける）。
- **X=2500**: 動く床（既存） + 落下スパイクの組み合わせ。

1. **FallingSpike作成**: `scenes/FallingSpike.tscn` & `scripts/FallingSpike.gd`.
2. **Main.tscn配置**: 上記要素を配置。
3. **検証**: ウォークスルー更新。
